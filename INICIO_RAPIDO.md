# ⚡ Inicio Rápido - 3 Pasos

## 1️⃣ Instala Python y la dependencia

```bash
# Descarga Python de python.org/downloads (si no lo tienes)

# Luego instala la dependencia:
pip install requests
```

## 2️⃣ Descarga este proyecto

**Opción fácil:** Descarga el ZIP desde GitHub y descomprímelo

**O con Git:**
```bash
git clone https://github.com/igfernandezlanda/claude-workspace.git
cd claude-workspace
```

## 3️⃣ Ejecuta el script

```bash
python discord_travel_tips.py
```

Y sigue las instrucciones en pantalla.
