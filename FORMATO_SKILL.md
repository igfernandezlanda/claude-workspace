# Formato de Tips de Viaje para Discord

## Estructura del Tip

```
{emoji} **{Nombre del Lugar}** {estrellas}
{separador}
📍 {Zona} • {Subsección}
🚇 {Metro} (opcional)
💰 {Precio}

✨ **Vibe:** {Descripción del ambiente}
🍽️ **Specialty:** {Especialidad del lugar}
💡 **Pro Tip:** {Consejo profesional}
```

## Emojis Custom por Categoría

| Categoría | Emoji | Separador | Color |
|-----------|-------|-----------|-------|
| Restaurantes | :Marcador1_v2: | ――― | Rojo (#E74C3C) |
| Cafés | :Marcador2_v2: | ╭─╮ | Naranja (#F39C12) |
| Bares | :Marcador3_v2: | ┌─┐ | Morado (#9B59B6) |
| Museos | :Marcador4_v2: | ═══ | Azul (#3498DB) |
| Parques | :Marcador5_v2: | ─── | Verde (#2ECC71) |
| Compras | :Marcador6_v2: | ━━━ | Rosa (#E91E63) |

## Sistema de Estrellas

- **4.9-5.0**: ⭐️⭐️⭐️
- **4.7-4.8**: ⭐️⭐️
- **4.5-4.6**: ⭐️
- **< 4.5**: (sin estrellas)

## Límites de Concisión

- **Vibe**: ≤ 15 palabras
- **Specialty**: ≤ 12 palabras
- **Pro Tip**: ≤ 20 palabras

## Niveles de Precio

- **€**: Económico
- **€€**: Moderado
- **€€€**: Caro
- **€€€€**: Muy caro

## Ejemplo Real

```
:Marcador1_v2: **La Carmencita** ⭐️⭐️⭐️
―――
📍 Malasaña • Centro
🚇 Tribunal
💰 €€€

✨ **Vibe:** Restaurante histórico con decoración belle époque y ambiente acogedor
🍽️ **Specialty:** Cocina española tradicional con toques modernos
💡 **Pro Tip:** Reserva con antelación, especialmente para cena los fines de semana
```

## Notas Importantes

1. Los emojis custom (:Marcador1_v2:, etc.) deben estar configurados en tu servidor de Discord
2. Los separadores deben coincidir exactamente con la categoría
3. Siempre validar los límites de palabras antes de publicar
4. La información de metro es opcional pero recomendada para ciudades con transporte público
5. Los ratings se obtienen automáticamente de Google Places API si está configurada
